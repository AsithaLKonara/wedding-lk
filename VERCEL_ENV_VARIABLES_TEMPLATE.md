## üéØ Step-by-Step Setup Process

### 1. MongoDB Atlas Setup
1. Go to https://cloud.mongodb.com/
2. Create cluster: `weddinglk-prod`
3. Create user: `weddinglk`
4. Whitelist IP: `0.0.0.0/0`
5. Get connection string
6. Replace `YOUR_PASSWORD` with actual password

### 2. Stripe Setup
1. Go to https://dashboard.stripe.com/
2. Switch to "Live mode"
3. Get live API keys
4. Set up webhook endpoint
5. Replace `your-production-stripe-*` with actual keys

### 3. Google OAuth Setup
1. Go to https://console.developers.google.com/
2. Create OAuth 2.0 credentials
3. Add redirect URI
4. Replace `your-production-google-*` with actual credentials

### 4. Redis Setup
1. Go to https://console.upstash.com/
2. Create Redis database
3. Get connection URL
4. Replace `your_token` and `your_endpoint` with actual values

### 5. Email Setup
1. Create Gmail account: `weddinglk.noreply@gmail.com`
2. Enable 2FA
3. Generate app password
4. Replace `your-production-email-app-password` with actual password

## üîí Security Notes

### Important Security Considerations:
1. **Never commit** environment variables to version control
2. **Use strong passwords** for all services
3. **Rotate keys regularly** for security
4. **Monitor access logs** for unusual activity
5. **Use HTTPS** for all connections

### Password Requirements:
- MongoDB: Minimum 12 characters, mixed case, numbers, symbols
- NextAuth Secret: Minimum 32 characters, random
- Email App Password: Generated by Google (16 characters)
- Redis Token: Generated by Upstash (32+ characters)
